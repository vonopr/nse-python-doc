nse-python docs¶

The documention of nse-python project.

Here we provide a collection of usage cases.

To get help on nseplot1d command, run it with --help option:

nseplot1d --help

Download sample data¶

We provide some sample data to demenstrate how nseplot1d works. You may download it with

mkdir -p data
curl -o example-data.zip --header 'Private-Token: yiuskGfW5Z31io2bmwgh'  http://tesla.parallel.ru/api/v4/projects/145/packages/generic/example-data/0.1.0/example-data.zip
unzip example-data.zip
mv -u example-data/* data/
mkdir -p pics
cd pics

Basic plot¶

To get a basic plot of timeseries contained in ../data/series/energy.dsq run:

nseplot1d  ../data/series/energy.dsq

This will produce an image energy.png inside your working directory pics:

Input and output files¶

To alter the name of the output provide -o option:

nseplot1d ../data/series/energy.dsq -o energy_alter.jpg

Multiple input files are also possible. In this case -o option is not applicable.

nseplot1d ../data/series/energy.dsq ../data/series/phys.dsq

nseplot1d ../data/series/*.dsq

Select variables¶

Choose variables to plot with --vars option:

nseplot1d ../data/series/energy.dsq  --vars "u-TKE [avg]" "v-TKE [avg]" -o energy-uv.png

X and Y bounds¶

You can specify X and Y bounds of plot with --t1, --t2 and vmin, vmax options. Units for X-bounds are record numbers, not physical time units.

nseplot1d --t1 50000 --t2 60000 --vmin 0.002 --vmax 0.004 ../data/series/energy.dsq -o energy-xy.png

Reference values¶

You can choose a reference variable within the input file with --ref option. Its values will be subtracted from all plotted variables:

nseplot1d ../data/series/energy.dsq --ref "TKE [avg]" --vars "TKE [avg]" "u-TKE [avg]" -o energy-tke-ref.png

You can choose a reference file with --diff option. Its values will be subtracted from all variables’ values of input files. If you additionall provide reference varaible with --ref option then values of reference variable in the reference file will be subtracted.

All input files must be conform with the reference file. They must have same variables and number of records.

nseplot1d ../data/series/energy.dsq --diff ../data/series/energy.dsq -o energy-tke-fref.png

Title, labels and legend¶

To add the title and axis labels use -t, --xlabel and --ylabel options

nseplot1d ../data/series/energy.dsq -t "TKEs, exp #1" --xlabel "time, hours" --ylabel "energy,  m²s⁻²"  -o energy-t.png

You can adjust the label’s position and orientation providing extra arguments to --xlabel and --ylabel. See --help for list of available keywords.

nseplot1d ../data/series/energy.dsq --ylabel "energy, m²s⁻²" top hor -t "TKEs, exp #1" --xlabel "time, hours" right   -o energy-t-ad.png

Legend position is contolled with --legend-loc option. See --help for list of available keywords.

nseplot1d ../data/series/energy.dsq --legend-loc="upper right" -t "TKEs, exp #1" --xlabel "time, hours"  --ylabel "energy, m²s⁻²"   -o energy-lloc.png
